# Components List

- **ServerSetup** – Handles the initial configuration of the server connection.
- **GameList** – Displays the list of available games and allows selecting one.
- **GameSetup** – Manages the setup phase for creating or joining a game.
- **GameInfo** – Displays the selected game’s details, including the game name and the list of connected players. Allows the user to decide whether to join the game or stay out.
- **Board** – Renders the main game board and its interactive elements.
- **DiceRoll** – Allows players to roll the dice during gameplay.
- **ItemSelection** – Lets players select and confirm required game items.
- **CallStatus** – Shows the current call state and each player’s response.
- **GameResult** – Displays the final outcome and summary of the game.
